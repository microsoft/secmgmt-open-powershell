# Disconnect-SecMgmtAccount

## SYNOPSIS
Disconnects the current session from the Microsoft cloud.

## SYNTAX

```powershell
Disconnect-SecMgmtAccount [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Disconnects the current session from the Microsoft cloud.

## EXAMPLES

### Example 1
```powershell
PS C:\> Disconnect-SecMgmtAccount
```

Disconnects the current session from the Microsoft cloud.
